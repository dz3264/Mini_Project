Database Introduction

The Movie lens dataset (ml-25m) describes rating and free-text tagging activity from MovieLens, a movie recommendation service. It contains 25000095 ratings and 1093360 tag applications across 62423 movies. These data were created by 162541 users between January 09, 1995 and November 21, 2019. This dataset was generated on November 21, 2019.
The Movie Database (TMDb) is a community built movie and TV database; we combined Movielens database with TMDb for rating information.
